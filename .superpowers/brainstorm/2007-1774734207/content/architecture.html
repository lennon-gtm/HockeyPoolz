Key architectural decisions baked in

  • Supabase Row-Level Security — each league's data is automatically isolated at the database level. League A can never see League B's data.
  • StatsService module boundary — one interface, one place. NHL API calls never leak into components or other APIs.
  • Vercel deployment — zero-config, scales automatically, free tier covers Alpha comfortably.
  • Real-time via Supabase channels — standings update live when scores change. No polling needed.
